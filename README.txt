# AnnotAPP

AnnotAPP is a desktop tool for managing text annotations, ideal for researchers, writers, and anyone needing quick access to text snippets.

## GitHub Repository

[https://github.com/DavidRA1025/AnnotAPP](https://github.com/DavidRA1025/AnnotAPP)

## Features

- Create and edit text annotations with custom names
- Two viewing modes: single and stacked annotations
- Import/export annotations to Excel files
- Adjust font size for better readability
- Always-on-top option for easy access
- Real-time character count
- Smart text selection for compound words and email addresses

## Installation

1. Download the AnnotAPP executable file from the [releases page](https://github.com/DavidRA1025/AnnotAPP/releases).
2. Place the file in your preferred folder.
3. Double-click to run the application.

Note: No installation process is required.

## Usage

- Click "Settings" to add, edit, or delete annotations.
- Use the toggle switch or Ctrl+M to switch between single and accumulative modes.
- Ctrl+L clears the display.
- Ctrl + Mouse Wheel adjusts font size.

## Importing Existing Data

1. Prepare an Excel file with columns "Name" and "Annotation".
2. Use the Import function in Settings to load your data.
